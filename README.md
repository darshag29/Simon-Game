🎮 Simon Game

A browser-based implementation of the classic Simon memory game built using HTML, CSS, JavaScript, and jQuery.

📖 About the Project

Simon Game is a memory challenge where players must repeat an increasingly long sequence of colored button presses. The game generates a random color pattern, and the player must accurately reproduce the sequence to advance through the levels.

As the game progresses, the sequence becomes longer, making it more difficult to remember and replicate. The game ends when the player selects an incorrect color.

🚀 Features

* Interactive gameplay with visual and audio feedback
* Random sequence generation
* Progressive difficulty with increasing levels
* Color animations and button press effects
* Game over detection and restart functionality
* Responsive browser-based interface

🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* jQuery

🎯 How to Play

1. Press any key to start the game.
2. Watch the highlighted color sequence.
3. Repeat the sequence by clicking the colored buttons.
4. Successfully reproduce the sequence to advance to the next level.
5. The game ends if you click the wrong color.
6. Press any key to restart after a game over.
